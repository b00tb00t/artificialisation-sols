# Artificialisation des Sols en France

Analyse de l'évolution de l'occupation du sol en France métropolitaine
à partir des données OCS GE (NOUVELLE GENERATION) et Admin Express

## Sources
- OCS GE NG — IGN
- Admin Express COG — IGN
- Populations — INSEE

## Stack technique
- R / RStudio
- Packages : sf, dplyr, tmap, ggplot2, tidyr, purrr
- Publication : Quarto → GitHub Pages
